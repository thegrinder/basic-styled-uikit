## **TypographyProvider**

`<TypographyProvider />` is responsible for passing down the theme to the `<Heading />` and `<Text />` component. It expects the theme to be of certain structure.


### **Minimal example**

The minimal typography theme has to include `baseFontSize`, `baseLineHeight`, `neutral` color, and `high` opacity value. You also need to provide the default `m` text size and the heading sizes you'll use (normally `h1` - `h6`).

<Playground>
  {() => {
    const typographyTheme = {
      baseFontSize: 16,
      baseLineHeight: 1.5,
      opacities: {
        high: 1,
      },
      colors: {
        neutral: '#161616',
      },
      text: {
        sizings: {
          m: {
            fontSize: '1rem',
          },
        },
      },
      heading: {
        sizings: {
          h1: {
            fontSize: '3rem',
          },
          h2: {
            fontSize: '2.5rem',
          },
        },
      },
    };

    const Example = () => (
      <TypographyProvider theme={typographyTheme}>
        <>
          <Heading as="h1">Neutral heading H1</Heading>
          <Heading as="h2">Neutral heading H2</Heading>
          <Text>Neutral color and size m text</Text>
        </>
      </TypographyProvider>
    );

    return <Example />;
  }}
</Playground>


However, you can easily add as many colors, opacities and sizes as you want. You can also add common style for heading or text under the `common` key.


<Playground>
  {() => {
    const typographyTheme = {
      baseFontSize: 16,
      baseLineHeight: 1.5,
      opacities: {
        high: 1,
        medium: 0.8,
        low: 0.6,
      },
      colors: {
        neutral: '#161616',
        primary: '#1e88e5',
        success: '#43a047',
        warning: '#fdd835',
        danger: '#f4511e',
      },
      text: {
        sizings: {
          l: {
            fontSize: '1.25rem',
          },
          m: {
            fontSize: '1rem',
          },
          s: {
            fontSize: '.875rem',
          },
          xs: {
            fontSize: '.75rem',
          },
        },
        common: {
          fontFamily: `'Inter var', sans-serif`,
        },
      },
      heading: {
        sizings: {
          h1: {
            fontSize: '3rem',
          },
          h2: {
            fontSize: '2.5rem',
          },
          h3: {
            fontSize: '2rem',
          },
          h4: {
            fontSize: '1.5rem',
          },
          h5: {
            fontSize: '1.25rem',
          },
          h6: {
            fontSize: '1rem',
          },
        },
        common: {
          fontFamily: `'Inter var', sans-serif`,
        },
      },
    };

    const Example = () => (
      <TypographyProvider theme={typographyTheme}>
        <>
          <Heading as="h1" color="primary">primary heading</Heading>
          <Heading as="h2" color="danger">danger heading</Heading>
          <Heading as="h3" color="success">success heading</Heading>
          <Heading as="h4" color="warning">warning heading</Heading>
          <Heading as="h5" color="primary">primary heading</Heading>
          <Heading as="h6">neutral heading</Heading>
          <Text emphasis="high">text neutral emphasis high</Text>
          <Text emphasis="medium">text neutral emphasis medium</Text>
          <Text emphasis="low">text neutral emphasis low</Text>
        </>
      </TypographyProvider>
    );

    return <Example />;
  }}
</Playground>


### **Theme with different backgrounds**

The color of the font usually depends on its background and that's why you can also account for that interdependency in the theme. You can define colors for the corresponding backgrounds specified in `<Box />` component.

<Playground>
  {() => {
    const boxTheme = {
      bgColors: {
        neutral: '#fff',
        primary: '#1e88e5',
        success: '#43a047',
        warning: '#fdd835',
        danger: '#f4511e',
      },
    };

    const typographyTheme = {
      baseFontSize: 16,
      baseLineHeight: 1.5,
      opacities: {
        high: 1,
      },
      colors: {
        neutral: '#161616',
        primary: '#1e88e5',
      },
      onBackground: {
        primary: {
          neutral: '#fff',
        },
        success: {
          neutral: '#fff',
        },
        warning: {
          neutral: '#161616',
        },
        danger: {
          neutral: '#fff',
        },
      },
      text: {
        sizings: {
          m: {
            fontSize: '1rem',
          },
        },
      },
      heading: {
        sizings: {
          h4: {
            fontSize: '1.5rem',
          },
        },
      },
    };

    const Example = () => (
      <BoxProvider theme={boxTheme}>
        <TypographyProvider theme={typographyTheme}>
          <>
            <Box bgColor="primary">
              <Heading as="h4">neutral heading on primary background</Heading>
            </Box>
            <Box bgColor="success">
              <Heading as="h4">neutral heading on success background</Heading>
            </Box>
            <Box bgColor="danger">
              <Text>neutral text on danger background</Text>
            </Box>
            <Box bgColor="warning">
              <Text>neutral text on warning background</Text>
            </Box>
            <Box bgColor="warning">
              <Heading as="h4" ignoreBackground={true} color="primary">
                background ignored - primary heading
              </Heading>
            </Box>
          </>
        </TypographyProvider>
      </BoxProvider>
    );

    return <Example />;
  }}
</Playground>
