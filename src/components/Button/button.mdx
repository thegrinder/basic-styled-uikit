## **ButtonProvider**

`<ButtonProvider />` is responsible for passing down the theme to the `<Button />` component. It expects the theme to be of certain structure. You can use any valid css properties written in camel case.


### **Minimal example**

The minimal button theme has to include `m` sizing and `default` button type.

<Playground>
  {() => {
    const buttonTheme = {
      btnTypes: {
        default: {
          normal: {
            /* you can put any valid css properties here */
            color: '#fff',
            backgroundColor: '#1e88e5',
          },
        },
      },
      sizings: {
        m: {
          /* you can put any valid css properties here */
          fontSize: '1rem',
          padding: '0.5rem 1rem',
        },
      },
    };

    const Example = () => (
      /* TypographyProvider is required for ButtonProvider to work */
      <TypographyProvider>
        <ButtonProvider theme={buttonTheme}>
          <Button>simple button</Button>
        </ButtonProvider>
      </TypographyProvider>
    );

    return <Example />;
  }}
</Playground>


### **Theme with different button states**

You can style different button states.

<Playground>
  {() => {
    const buttonTheme = {
      btnTypes: {
        default: {
          normal: {
            color: '#fff',
            backgroundColor: '#1e88e5',
          },
          hover: {
            backgroundColor: '#3594e8',
          },
          active: {
            backgroundColor: '#187bd1',
          },
        },
      },
      sizings: {
        m: {
          fontSize: '1rem',
          padding: '0.5rem 1rem',
        },
      },
    };

    const Example = () => (
      <TypographyProvider>
        <ButtonProvider theme={buttonTheme}>
          <Button>hover and click</Button>
        </ButtonProvider>
      </TypographyProvider>
    );

    return <Example />;
  }}
</Playground>


### **Theme with different button types, sizings and common style**

Also, you can add different button types and sizings.

<Playground>
  {() => {
    const buttonTheme = {
      btnTypes: {
        default: {
          normal: {
            color: '#fff',
            backgroundColor: '#1e88e5',
          },
          hover: {
            backgroundColor: '#3594e8',
          },
          active: {
            backgroundColor: '#187bd1',
          },
        },
        danger: {
          normal: {
            color: '#fff',
            backgroundColor: '#f4511e',
          },
          hover: {
            backgroundColor: '#f56436',
          },
          active: {
            backgroundColor: '#ed410c',
          },
        },
      },
      sizings: {
        m: {
          fontSize: '1rem',
          padding: '0.5rem 1rem',
        },
        l: {
          fontSize: '1.5rem',
          padding: '0.75rem 1.25rem',
        },
      },
      common: {
        fontWeight: '600',
        fontFamily: `'Inter var', sans-serif`,
      },
    };

    const Example = () => (
      <TypographyProvider>
        <ButtonProvider theme={buttonTheme}>
          <>
            <Button className="margin-bottom">default button type</Button>
            <Button btnType="danger" className="margin-bottom">
              danger button type
            </Button>
            <Button className="margin-bottom">default sizing m</Button>
            <Button sizing="l" className="margin-bottom">
              sizing l
            </Button>
          </>
        </ButtonProvider>
      </TypographyProvider>
    );

    return <Example />;
  }}
</Playground>

## **Button**

Button component grabs the theme given to `<ButtonProvider />` and applies the style defined there based on the given `btnType` and `sizing` prop value. It also accepts a few different non-theme related props for convenience reasons.
