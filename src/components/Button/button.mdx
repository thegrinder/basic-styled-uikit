## **ButtonProvider**

`<ButtonProvider />` is responsible for passing down the theme to the `<Button />` component. It expects the theme to be of certain structure.


---


## **Button**

Button component grabs the theme given to `<ButtonProvider />` and applies the style defined there based on the given `btnType` and `sizing` prop values. It also accepts a few different non-theme related props for convenience reasons.

<Props of={Button} />
