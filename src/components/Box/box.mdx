## **BoxProvider**

`<BoxProvider />` is responsible for passing down the theme to the `<Box />` component. It expects the theme to be of certain structure. You can use any valid css properties written in camel case.


### **Example**

The minimal color box theme has to include one background color and one elevation variation.

<Playground>
  {() => {
    const boxTheme = {
      bgColors: {
        primary: '#1e88e5',
      },
      elevations: {
        1: {
          boxShadow:
            '0 3px 1px -2px rgba(0,0,0,.2), 0 2px 2px 0 rgba(0,0,0,.14), 0 1px 5px 0 rgba(0,0,0,.12)',
        },
      },
    };

    const Example = () => (
      <BoxProvider theme={boxTheme}>
        <>
          <Box bgColor="primary">background color primary</Box>
          <Spacer />
          <Box elevation={1}>elevation one</Box>
        </>
      </BoxProvider>
    );

    return <Example />;
  }}
</Playground>


However, you can easily add as many background colors and elevation variations as you want.

<Playground>
  {() => {
    const boxTheme = {
      bgColors: {
        neutral: '#fff',

        success: '#43A047',

      },
      elevations: {
        1: {
          boxShadow:
            '0 3px 1px -2px rgba(0,0,0,.2), 0 2px 2px 0 rgba(0,0,0,.14), 0 1px 5px 0 rgba(0,0,0,.12)',
        },
        2: {
          boxShadow:
            '0 2px 4px -1px rgba(0,0,0,.2), 0 4px 5px 0 rgba(0,0,0,.14), 0 1px 10px 0 rgba(0,0,0,.12)',
        },
      },
    };

    const Example = () => (
      <BoxProvider theme={boxTheme}>
        <>

          <Box bgColor="primary">
            primary
          </Box>
          <Spacer />
          <Box bgColor="danger">
            danger
          </Box>
          <Spacer />
          <Box bgColor="success">
            success
          </Box>
          <Spacer />
          <Box bgColor="warning">
            warning
          </Box>
          <Spacer />
          <Box bgColor="neutral">
            neutral
          </Box>
          <Spacer />

          <Box bgColor="neutral" style={{ padding: '20px' }}>
            <Box elevation={1} style={{ padding: '20px' }}>
              <span>one</span>
              <Box elevation={1} style={{ padding: '20px' }}>
                <span>one</span>
                <Box elevation={1} style={{ padding: '20px' }}>
                  <span>one</span>
                </Box>
              </Box>
            </Box>
            <Spacer />
            <Box elevation={2}>
              <span>two</span>
            </Box>
          </Box>

        </>
      </BoxProvider>
    );

    return <Example />;
  }}
</Playground>

## **Box**

Box component grabs the theme given to [BoxProvider](/basic-styled-uikit/components/box#BoxProvider) and applies the style defined there based on the given `bgColor` and `elevation` prop value.
