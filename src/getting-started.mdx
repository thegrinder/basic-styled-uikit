# Getting started

## Installation

Basic-styled-uikit is [available on npm](https://www.npmjs.com/package/basic-styled-uikit) and it can be installed using npm or yarn.

```
# Using npm
npm install basic-styled-uikit --save

# Using Yarn
yarn add basic-styled-uikit
```


## Quick setup

Basic-styled-uikit uses styled-components under the hood and that's why you have to wrap your application with a few theme providers responsible for passing style down to its respective copmonents. The quickest way to get going is to install and use the default theme.

```
# Using npm
npm install basic-styled-uikit-theme --save

# Using Yarn
yarn add basic-styled-uikit-theme
```

```js
import React from 'react';

// basic-styled-uikit providers
import {
  ModeProvider,
  TypographyProvider,
  ButtonProvider,
  ColorBoxProvider,
  LinkProvider,
  FormProvider,
  ColorsProvider,
} from 'basic-styled-uikit';

// respective themes from basic-styled-uikit-theme
import {
  typographyTheme,
  buttonTheme,
  colorBoxTheme,
  linkTheme,
  formTheme,
  colorsTheme,
} from 'basic-styled-uikit-theme';


// root component
const App = ({ children }) => (
  <ModeProvider>
    <TypographyProvider theme={typographyTheme.base} darkTheme={typographyTheme.dark}>
      <ColorBoxProvider theme={colorBoxTheme} darkTheme={colorBoxTheme.dark}>
        <ButtonProvider theme={buttonTheme} darkTheme={buttonTheme.dark}>
          <LinkProvider theme={linkTheme} darkTheme={linkTheme.dark}>
            <FormProvider theme={formTheme} darkTheme={formTheme.dark}>
              <ColorsProvider theme={colorsTheme} darkTheme={colorsTheme.dark}>
                <>{children}</>
              </ColorsProvider>
            </FormProvider>
          </LinkProvider>
        </ButtonProvider>
      </ColorBoxProvider>
    </TypographyProvider>
  </ModeProvider>
);

App.propTypes = propTypes;

export default App;
```


## Custom setup

The main reason basic-styled-uikit was created was to have totally customizable and themeable base components that you can build on top of. That's why you can pack virtually any style properties into each of the themes. You can just pick the providers/components you need and ignore the rest. Please refer to the documentation of each of them for more information.

```js
import React from 'react';

// basic-styled-uikit providers
import {
  // the only required provider
  TypographyProvider,
  // providers you want to use
  ModeProvider,
  ButtonProvider,
  FormProvider,
} from 'basic-styled-uikit';

import {
  customTypographyTheme,
  customButtonTheme,
  customFormTheme,
} from 'path/to/your/custom/theme';


// root component
const App = ({ children }) => (
  <ModeProvider>
    {/* TypographyProvider is required */}
    <TypographyProvider theme={customTypographyTheme.base} darkTheme={customTypographyTheme.dark}>
      <ButtonProvider theme={customButtonTheme} darkTheme={customButtonTheme.dark}>
        <FormProvider theme={customFormTheme} darkTheme={customFormTheme.dark}>
          <>{children}</>
        </FormProvider>
      </ButtonProvider>
    </TypographyProvider>
  </ModeProvider>
);

App.propTypes = propTypes;

export default App;
```
